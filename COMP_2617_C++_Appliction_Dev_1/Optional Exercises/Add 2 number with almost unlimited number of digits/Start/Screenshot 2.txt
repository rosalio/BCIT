This is an example of adding two numbers where MAX_DIGITS is set to 30.

==========================================================================

Enter in the first number (max 30 digits):
12345678901234567890

Enter in the second number (max 30 digits):
1234567890123456789012345

The result of adding the 2 numbers input is:

           12345678901234567890
+     1234567890123456789012345
===============================
      1234580235802358023580235
