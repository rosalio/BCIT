Background Images

The tutorial teaches how to add background images in a web page. The CSS background properties are used to define the background effects of an element.

The CSS properties used for background effects can be:

  • background-color
  • background-image
  • background-repeat
  • background-position

The value of background-repeat can be:

  • repeat
  • repeat-y
  • repeat-x
  • no-repeat

The value of background-position can be:

  • top
  • center
  • bottom
  • left
  • right

To explore more details about these properties, you may visit the tutorial website: HTML Dog: Background Images
